# Nexus MACBETH — Elicitação Qualitativa e Programação Linear

Este repositório é o módulo independente do ecossistema **NEXUS MCDM**, configurado para operar exclusivamente com o método **MACBETH (Measuring Attractiveness by a Categorical Based Evaluation Technique)**.

---

## 🎨 Identidade Visual e Branding
- **Nome Oficial:** Nexus MACBETH
- **Cores Oficiais:** Âmbar (`#F59E0B`) e Azul Escuro (`#1E3A8A`)
- **Conceito Visual:** Blocos de intervalos de atratividade qualitativa representando a escala ordinal-cardinal.
- **Copyright:** Direitos Reservados © 2026 NEXUS-MCDM.

---

## 🌟 Recursos e Matemática

- **Julgamento Semântico:** Permite comparações de atratividade entre critérios utilizando categorias verbais (Nula, Muito Fraca, Fraca, Moderada, Forte, Muito Forte, Extrema).
- **Programação Linear:** Converte a consistência ordinal em uma escala cardinal de utilidade intervalar através de restrições de desigualdades resolvidas via LP (Programação Linear).
- **Relatório PDF & Excel TCC:** Exportação de relatório PDF.
